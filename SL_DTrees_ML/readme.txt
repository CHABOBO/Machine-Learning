Optical Recognition of Handwritten Digits dataset

1)The codebase for Optcial recogonition is present inside the folder DigitRecogonition.
2)DigitReco.R contains all the code for Optical Recogonition.
3)Input files and output files are also present in the same folder. 

4)Libraries Used:
rpart, rattle, rpart.plot, party, RColorBrewer

5)For running the program use command source source("DigitReco.R")
make sure that the working directory is inside the DigitRecogonition by using setwd() command
